Задача

Представители компании запросили найти удобное решение для консолидации коммуникаций компании и сохранения всей истории. Еще одним запросом стало разработать решение для экономии времени, обновления некоторых функции и составление ролей сотрудников компании с жестким разграничениями по доступам.

У клиента была одна тестовая задача — создать модуль для ограничения загрузки документа по IP, после решения которой началось наше двухлетнее сотрудничество.

Решение

Для решения поставленных задач компания установила собственный хостинг Bitrix24 из-за его многочисленных возможностей и гибкости функционала.

Мы разработали решение для сохранения безопасности корпоративных документов.

Когда сотрудник выходит из офиса, модуль ограничивает всё CRM файлы для выгрузки и загрузки. Разработка была реализована на нашем сервере, потому что WERTGARANTIE Group нуждалась в 100% конфиденциальности, а после окончания работ проект был загружен в систему компании с помощью сотрудника в локальном офисе в Германии.
